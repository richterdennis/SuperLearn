Was ist SuperLearn

SuperLearn ist ein im Rahmen des Moduls: „Praktikum mobile Anwendungen“ an der Technischen Hochschule Mittelhessen entstandenes Projekt. Ziel war es eine Lern-App für das Android Betriebssystem zu schaffen, welche Studenten beim lernen für das Studium hilft.

Was kann SuperLearn

SuperLearn bietet Studenten eine unabhängige Lernplattform. Die Studenten können sich mit ihrer Uni-Emailadresse anmelden und finden die Module ihres Studiengangs nach Semestern sortiert oder können neue Module hinzufügen. Wird ein Modul ausgewählt, so startet eine Spielrunde mit 10 Fragen zu dem Modul, die nacheinander abgefragt werden.

Warum SuperLearn

Jeder Student kann Fragen hinzufügen, sie positiv oder negativ Bewerten oder auch melden. Durch das Einstellen gutbewerteter Fragen und das Bewerten von Fragen anderer kann ein Student Medallien sammeln, welche auf seiner Profilseite angezeigt werden. Auch der Lernfortschritt einzelner Fragen und Module kann jederzeit eingesehen werden.
